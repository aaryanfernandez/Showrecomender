Show Recommender 

A python script that recomends tv shows and movies

Features
Stores watch history in a json file.
Recommends shows based on previously watched shows. 
Simple command- line interface
Easy to modify and expand

How to run 
Make sure pyhton is installed. 
Download or clone this repositiory. 
Run python main.py
